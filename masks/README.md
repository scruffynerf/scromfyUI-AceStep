## Inverting Masks
If you have a set of masks that need inverting (e.g., swapping black and white), you can use the provided utility script found in this directory.

1. You do need Pillow:  pip install pillow
2. Run the inversion script on your target directory:
   ```bash
   python3 invert_png.py your_folder
   ```
   This will create an `inverted` folder inside your target directory containing the inverted images.
